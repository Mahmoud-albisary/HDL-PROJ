# Basys3 with SystemVerilog
This is a repository for FPGA programming for educational purposes.
 
In this repository, the Basys3 development board is used. It is based on the Artix-7 FPGA and is widely used for educational purposes. For more information, check the following link: https://digilent.com/reference/_media/basys3:basys3_rm.pdf.

## AI Assistance
As we can see today, LLMs have changed the way we work on projects, and the projects in this repository are no exception. I would like to clarify that I am the one responsible for the project ideas and approach, but I used the help of ChatGPT in the following:

- Code debugging in some cases
- Some tasks that are not directly related to FPGA programming, such as writing the common directories and file names that are usually ignored in the `.gitignore` file
- Text generation for some parts of the provided documentation

The main aim is to make the process faster and easier, not to rely on AI, which would contradict the real purpose of the repository.
